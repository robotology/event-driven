Description

The module calculates event-based Optical Flow by fitting planes in spatio-temporal space.

Parameters

  • – name : Specifies the stem name of ports created by the module.
  • – strict : Sets both input and ouput ports to use strict protocols.
  • – width : Number of pixels on the x-axis of the sensor.
  • – height : Number of pixels on the y-axis of the sensor.
  • – filterSize : Lenght of the spatial window in pixels.
  • – minEvtsThresh : Minimum number of events on the plane.

Input Ports

  • /vFlow/vBottle:i [eventdriven::vBottle] [default carrier:tcp]: Accepts the address events in the vBottle container

Output Ports

  • /vFlow/vBottle:o [eventdriven::vBottle] [default carrier:tcp]: Outputs flow events in the form of an eventdriven::FlowEvent. The vBottle also contains all events in the vBottle received as input.
